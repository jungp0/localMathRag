# LocalMathRAG MTEF Parser

This directory vendors the MTEF v5 parser from
`github.com/zhexiao/mtef-go` at commit
`16d536beaf41e0d88b0963a59e55c36de54e4fba` (Apache-2.0).

`main.go` is a minimal LocalMathRAG wrapper. It accepts an Equation Editor OLE
object on stdin (or with `-f`) and writes the translated LaTeX to stdout.

The checked-in Linux AMD64 binary is generated by
`scripts/build-mtef-parser.ps1`. RAGFlow uses the binary locally, so document
parsing does not require Go, network access, an OCR model, or a running LLM.
